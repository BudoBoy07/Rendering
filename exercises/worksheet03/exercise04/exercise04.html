  • Exercise 4: Describe how texture aliasing is affected by pixel subdivision level and nearest versus linear filtering, respectively.
    Answer: Higher pixel subdivision levels reduce aliasing by increasing the number of sub-pixel samples. This produces smoother results. Nearest filtering causes more aliasing, as it picks the closest texel, leading to unclear/blocky artifacts. Linear filtering mitigates this by interpolating between texels. This gives smoother transitions and less aliasing. Scaling the texture amplifies aliasing in nearest filtering but is better for linear filtering (at higher subdivision levels).
